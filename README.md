# Autonomous-Drone-Medical-Transport
Autonomous Drone System for Emergency Medical Transportation using GPS, sensors and flight controller
🚁 Autonomous Drone System for Emergency Medical Transportation

An autonomous drone platform designed to deliver critical medical supplies such as blood units, vaccines, and emergency medicines to remote or traffic-congested areas during emergencies.

📌 Overview

This system uses GPS-based navigation, real-time telemetry, and onboard sensors to ensure safe and efficient medical delivery. It minimizes response time and operates with minimal human intervention.

⚙️ Key Features

Autonomous takeoff and landing

GPS waypoint navigation

Obstacle detection and avoidance

Real-time tracking via Ground Control Station

Return-to-Home (RTH) safety mechanism

Secure medical payload compartment

🧠 Technologies Used

Flight Controller

GPS Module

IMU Sensors (Gyroscope & Accelerometer)

Brushless Motors & ESC

Telemetry Module

🎯 Application

Useful in disaster situations (floods, earthquakes), rural healthcare delivery, and emergency medical logistics.
